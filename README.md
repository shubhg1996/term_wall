#Terminal wallpaper change
Requires:
Python >= 3.5
Terminology, Tilix or ITerm on MacOS

Instructions:
chmod a+x main.py
<Make directory "bin" in home and append to $PATH in .bashrc or .zshrc>
ln -s main.py ~/bin/twall

Usage:
twall <part of filename>
