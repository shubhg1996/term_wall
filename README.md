# Terminal wallpaper change
## Requires:
Python >= 3.5
Terminology, Tilix or ITerm on MacOS

## Instructions:
1. `chmod a+x main.py`
2. Make directory "bin" in home and append to $PATH in .bashrc or .zshrc
3. `ln -s main.py ~/bin/twall`

## Usage:
twall "part of filename"
